I'm feeling adventurous this week, and diving into sails.js

The rest of this post assumes that you already have node/npm installed.

Install sails.js
{% highlight bash %}
sudo npm -g install sails
{% endhighlight %}

Create our Project
{% highlight bash %}
sails new invenpuppy
cd invenpuppy
sails lift
{% endhighlight %}

Great, our app is running on http://localhost:1337/. Lets customize the homepage, and add the libraries we will need for the front-end: Bootstrap 3, Angular.js, and Jquery. Change your home/index.ejs to match the following - an adorable 2-column layout, complete with a puppy.

{% highlight html %}
<!doctype html>
<html ng-app='InvenPuppy'>
<head>
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1.0" />
<title>InvenPuppy</title>
<link rel="stylesheet" href="//netdna.bootstrapcdn.com/bootstrap/3.0.0/css/bootstrap.min.css">

<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.0.8/angular.min.js"></script>
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script src="//netdna.bootstrapcdn.com/bootstrap/3.0.0/js/bootstrap.min.js"></script>
</head>
<body>
<div class="container">
<div class="row">
<div class="brand col-md-3">
<a id="logo" href="/"><img src="/images/dog-256.png"></a>
</div>
<div class="col-md-9">
<div ng-view></div>
</div>
</div>
</div>
</body>
</html>
{% endhighlight %}

sails generate puppy
