• The successful candidate will be called upon to lead the direction and solution of the REST API implementation across all enterprise level applications.

• Node.js

• RESTful API Integration
• Working with large database tables

Preferred Experience
• oData Proficiency

• C# Development experience - .(NET 3.0, 4.0, 4.5) with API technologies
